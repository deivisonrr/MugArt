# MugArt - Caneca padrão 3D

Versão com caneca modelada em formato realista de caneca padrão 11oz:
- corpo levemente cônico;
- borda superior arredondada;
- base menor;
- interior visível;
- alça oval proporcional;
- arte aplicada em área curvada frontal;
- funciona no GitHub Pages.

Para publicar, envie `index.html`, `style.css`, `script.js` e a pasta `assets` para o repositório.
