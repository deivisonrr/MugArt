# MugArt Studio

Site/estúdio de personalização de canecas para GitHub Pages.

## Publicação rápida no GitHub Pages

1. Extraia o ZIP.
2. Abra a pasta `mugart_studio_v1`.
3. Envie para o repositório do GitHub:
   - `index.html`
   - pasta `assets`
   - pasta `css`
   - pasta `js`
4. No GitHub: `Settings > Pages`.
5. Configure:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
6. Salve e aguarde 1 a 3 minutos.

## Importante

O mockup 3D usa Three.js carregado por CDN. Portanto:
- no GitHub Pages funciona normalmente;
- abrir o HTML direto com duplo clique pode falhar em alguns navegadores.
