# MugArt Site Estático

Site estático com layout inspirado na referência enviada, sem 3D.

## Como publicar no GitHub Pages

1. Envie `index.html`, as pastas `css`, `js` e `assets` para a raiz do repositório.
2. No GitHub: Settings > Pages.
3. Source: Deploy from a branch.
4. Branch: main / root.
5. Aguarde atualizar e use Ctrl + F5.

## Personalização

- Troque as imagens em `assets/` por fotos reais das suas canecas quando tiver.
