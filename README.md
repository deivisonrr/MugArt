# MugArt Studio Pro - GLB Ready

Projeto estático para GitHub Pages com personalizador 3D de canecas.

## Como publicar

Envie estes arquivos para o repositório GitHub Pages:

- index.html
- css/style.css
- js/app.js
- assets/logo.png
- assets/models/  

## Modelo 3D profissional

Para qualidade semelhante a plataformas profissionais de mockup, coloque um modelo 3D licenciado neste caminho:

```text
assets/models/mug.glb
```

O site tenta carregar esse arquivo automaticamente. Se ele não existir, usa uma caneca 3D própria gerada em Three.js como fallback.

## Importante

Não foi usado código, modelo ou asset proprietário do Rapid Mockup. A aplicação está preparada para receber um modelo 3D próprio/licenciado da MugArt.
